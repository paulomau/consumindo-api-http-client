# consumindo-api-http-client
Exemplo de consumo de uma api com .net core c#

* A proposta deste código é apresentar o uso do HttpClient. 
* A codificação é simples e tem o propósito didático quanto à uma aplicação do recurso.
* Este código é anterior à nova implementação feita no .Net Core 5.0, para consumo de API em uma aplicação com C#.
